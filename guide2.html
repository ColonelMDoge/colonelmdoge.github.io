How To Create A Minecraft Server

Who Should Read This Guide?

From personal experience, hosting Minecraft servers from free online platforms were always a nuisance. Slower than desired, underperforming, barely any control, and allocated resources were very scarce. Instead, if you created a Cloud Compute Instance through OCI, you can get around the rabbit hole and create something far more powerful.

Prerequsite: Installing TMUX On Your Server

TMUX is a application that can be installed through your SSH. Its major use is that it can create "headless windows" that act as separate secure shell hosts. It is very useful because these windows persist after the closure of the secure shell (in this case, Putty) that connects with the server. Multiple of these windows can be run simultaneously, too. TMUX is what elevates a localhost only server to a dedicated 24/7 server that is playable anytime.

To install TMUX, go to the terminal of the SSH and execute this command:

sudo yum install tmux

Enter "y" after the prompt and TMUX should be installed onto your instance.

Convenient TMUX Commands

Action Command
Creating a new session 
tmux new -s [SESSION NAME]
Killing a session 
tmux kill-ses -t [SESSION NAME]
Attaching to a session 
tmux a -t [SESSION NAME]
Listing sessions 
tmux ls
Exiting a session 
Ctrl B + D
Renaming a session 
Ctrl B + $

NOTE: More information can be found here at https://tmuxcheatsheet.com/
